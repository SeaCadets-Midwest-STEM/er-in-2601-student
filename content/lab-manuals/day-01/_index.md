On Day 1, you'll write your first firmware programs and build circuits that bring the microcontroller to life.

## Sessions

### Session 1B - [Hello Matrix: Your First Program](/lab-manuals/day-01/hello-matrix/)
Wire an LED to your Arduino Uno Q and write a complete program from scratch to make it blink, then experiment with different patterns.

### Session 1C - [LED Chase Pattern: Loops That Do the Work](/lab-manuals/day-01/led-chase/)
Make three LEDs "chase" each other using a `for` loop - and discover how one line of code can control multiple pins.

### Session 1D - [Button Input: If/Else Decision Making](/lab-manuals/day-01/button-input/)
Give your Arduino its first sense. Wire a pushbutton and write code that makes the board change behavior based on whether the button is pressed.
