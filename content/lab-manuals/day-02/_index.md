On Day 2, you'll learn how microcontrollers measure the real world. You'll build voltage dividers, read analog sensor data, and discover why real-world signals need special handling.

## Sessions

### Session 2B - [Light Sensor Circuit](/lab-manuals/day-02/light-sensor-circuit/)
Build a light-sensing circuit using an LDR (light-dependent resistor) in a voltage divider configuration. Read analog values through the Arduino's ADC and use the sensor to control an LED's brightness.

### Session 2C - [Potentiometer and Analog Reading](/lab-manuals/day-02/potentiometer-analog/)
Explore how a potentiometer works as a manually controlled variable resistor. Practice using `map()` to translate sensor ranges to useful output ranges.

### Session 2D - [Sensor Thresholds and Hysteresis](/lab-manuals/day-02/hysteresis/)
Discover why sensor readings near a threshold cause flickering and learn the technique called **hysteresis** that fixes it.

### Session 2F - [Sensor Scavenger Hunt Challenge](/lab-manuals/day-02/sensor-scavenger-hunt/)
Open-ended team challenge: design and build a sensor-based device that reacts to its environment. Combines everything from Sessions 2B-2D into one creative project with a demo phase.
