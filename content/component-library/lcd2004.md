## What Is It?

A blue-backlit screen that shows 4 lines of 20 characters each. It uses the standard HD44780 controller, which means Arduino has built-in support for it via the `LiquidCrystal` library.

The 10-pin cable (15687-2) makes wiring easy: plug one end into the LCD's header pins and the other end into your breadboard.

## When to Use It

- Showing sensor readings (temperature, distance, etc.)
- Displaying menus or status messages
- Any project where you need to read more than a single word or number

## Pinout (on the cable)

The 10-pin cable connects to the LCD module. The pins on the cable's connector are:

| Pin # | Name | Connect To |
|---|---|---|
| 1 | `VSS` | Arduino `GND` |
| 2 | `VDD` | Arduino `5V` |
| 3 | `VEE` | Potentiometer wiper (controls contrast) |
| 4 | `RS` | Arduino digital pin `D12` |
| 5 | `RW` | Arduino `GND` (write-only mode) |
| 6 | `E` | Arduino digital pin `D11` |
| 7 | `D0` | (Not used — 4-bit mode) |
| 8 | `D1` | (Not used — 4-bit mode) |
| 9 | `D2` | (Not used — 4-bit mode) |
| 10 | `D3` | Arduino digital pin `D10` |
| 11 | `D4` | Arduino digital pin `D9` |
| 12 | `D5` | Arduino digital pin `D8` |
| 13 | `D6` | Arduino digital pin `D7` |
| 14 | `D7` | Arduino digital pin `D6` |
| 15 | `A` (backlight +) | Arduino `5V` (through a 220Ω resistor) |
| 16 | `K` (backlight −) | Arduino `GND` |

> **Note:** Pins 1–14 connect to the LCD data/control header. Pins 15–16 power the backlight. Check your cable labeling — some cables combine pins differently.

### Contrast Potentiometer

Pin 3 (`VEE`) controls text contrast. Connect it to the wiper (middle pin) of a potentiometer. The outer pins go to `5V` and `GND`. Turn the pot until the text is clearly visible.

## Example Code

```cpp
#include <LiquidCrystal.h>

// RS=E12, E=D11, D4=D9, D5=D8, D6=D7, D7=D6
LiquidCrystal lcd(12, 11, 9, 8, 7, 6);

void setup() {
  lcd.begin(20, 4);          // 20 columns, 4 rows
  lcd.print("Hello, World!");
}

void loop() {
  lcd.setCursor(0, 1);       // column 0, row 1 (second line)
  lcd.print(millis() / 1000);
  lcd.print(" sec running");
  delay(100);
}
```

## Common Pitfalls

- **No text visible?** Adjust the contrast potentiometer. This is the #1 issue.
- **Garbled characters?** Check that pin 3 (`VEE`) is connected and that the data pins match your code.
- **Backlight too bright?** Add a 220Ω resistor in series with pin 15 (`A`).
- **Library needed?** The `LiquidCrystal` library comes pre-installed with Arduino IDE.
