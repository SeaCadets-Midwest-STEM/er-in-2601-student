## What Is It?

An LDR is a resistor that changes its resistance based on how much light hits it:

- **Bright light** → low resistance (electricity flows easily)
- **Darkness** → high resistance (electricity flows poorly)

The Arduino reads this change as a different voltage on an analog pin.

## When to Use It

- Automatic night light (turn on LED when it gets dark)
- Day/night detector
- Light meter
- Sun-tracking projects
- any project that needs to react to ambient light

## Pinout

The LDR has 2 legs (it's a resistor, so direction doesn't matter):

| Connection | Where |
|---|---|
| LDR leg 1 | Arduino `5V` |
| LDR leg 2 | Arduino analog pin (e.g., `A0`) **AND** one end of a 10kΩ resistor |
| 10kΩ resistor other end | Arduino `GND` |

This forms a **voltage divider**: the LDR and the resistor share the 5V, and the Arduino reads the voltage at the point between them.

## How It Works (Simple)

The voltage divider creates a voltage at `A0` that changes with light:

- **Bright light:** LDR resistance drops → voltage at `A0` goes UP → `analogRead()` returns a HIGH number (close to 1023)
- **Dark:** LDR resistance rises → voltage at `A0` goes DOWN → `analogRead()` returns a LOW number (close to 0)

**There is no standard "light unit"** — the values are relative. You'll need to observe the range in your environment and pick a threshold that works.

## Example Code

```cpp
const int ldrPin = A0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int lightLevel = analogRead(ldrPin);
  Serial.print("Light level: ");
  Serial.println(lightLevel);

  // Example: turn on LED if it's dark
  if (lightLevel < 300) {
    Serial.println("It's dark!");
  }

  delay(500);
}
```

## Common Pitfalls

- **No resistor = no reading:** The LDR MUST be part of a voltage divider. Connecting it directly to 5V and an analog pin won't work.
- **Threshold varies by environment:** A value of 300 might mean "dark" in a bright room but "bright" in a dim hallway. Test in your actual environment.
- **Slow response:** LDRs take a fraction of a second to adjust. They're great for ambient light, not for detecting fast flashes.
- **Leg direction doesn't matter:** The LDR is a resistor — either leg can go to 5V or to the resistor.
